## Weekly Sunday review (15 min, every Sunday)

1. Open GBP Insights — note this week's profile views, searches, calls
2. Open IG analytics — note follower growth, reach, top post
3. Count new reviews this week
4. Map pack check: Google "microblading Makati" in incognito, note position
5. Update tracking spreadsheet
6. Plan next week's content if not already scheduled

## Monthly review (30 min, last Sunday of each month)

1. Full GBP performance vs last month
2. IG growth + best-performing posts
3. New citations added
4. Reviews received vs target
5. What worked, what didn't, what to change
6. Adjust month-ahead plan
