# Citations Execution Playbook (for the Helper)

File 06 is the list of where to submit. This file is how to actually do it: the exact text to paste, the order, and the tracker. Goal: consistent NAP mentions of Kate Kanner Permanent Makeup across local Philippine websites to push GBP map-pack ranking and feed AI assistants (ChatGPT, Copilot, Siri all read these directories).

Total effort: ~8 hours spread over 4 weeks. Do not rush it in one day; 6-10 new listings per week looks natural, 31 in one day looks like spam.

## Hard rules

- NAP is sacred: copy-paste only, never retype, never "improve" the address
- Business name never gets keywords bolted on ("Kate Kanner Permanent Makeup Makati Best Brows" = penalty risk)
- Skip any site demanding payment or a credit card for a basic listing
- Skip "500 directories for $20" services (file 06 list of what to avoid)
- One submission account/email everywhere = recoverable forever
- When Kate changes clinics: only the address line changes, everything else stays (this is why citations live under her name, not Reverie's)
